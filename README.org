* Howto
To use the dotfiles, 

1) tangle the documents. The dotfiles will be created in
   [[./dotfiles/]]. If the directory does not exist, it will be created
   automatically.
2) create in the directory where the dotfiles should be a link to the
   tangled file in [[./dotfiles/]]
